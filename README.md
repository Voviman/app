# Spendings application

This app allows users to track Spendings.

#### Intructions 
1.Download the source code

```bash
git clone https://github.com/adoublebOS/webtech_refer.git
```
or just download the *.zip file

2.Install deps
```bash
npm install
```

3.Run the app
```bash
node app
```

#### Deps list
- express
- pug
